# US-022 — Gestion du profil utilisateur

## Statut

🟡 Ready for Implementation

---

## Description

Permettre à un utilisateur authentifié de modifier les informations de son profil personnel.

Cette User Story complète **US-021 – Consultation du profil utilisateur**.

---

## Objectifs

- Modifier les informations personnelles.
- Garantir la cohérence des données utilisateur.
- Offrir une expérience fluide et sécurisée.
- Préparer l'utilisateur à devenir professionnel.
