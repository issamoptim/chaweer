## Référence

La User Story constitue l'unique source de vérité concernant :

- les règles métier ;

- le périmètre fonctionnel.

Ce dossier ne les duplique pas.

---

# Objectif

Fournir tous les éléments nécessaires à la conception, à l'illustration et à l'implémentation de la fonctionnalité **Sécurité du compte**.

Les livrables de ce dossier sont destinés :

- au Product Designer ;
- au Frontend Developer ;
- au Backend Developer ;
- aux assistants IA (Claude, Devin, Cursor, Codex…).
