## Description

As a lawyer, I want a guided multi-step onboarding wizard so that I can complete all registration steps in a logical, user-friendly sequence.

## Acceptance Criteria (Draft)

- Multi-step wizard connecting: registration (US-028) → credentials (US-029) → documents (US-030) → specializations & languages (US-031) → initial profile (US-032)
- Progress indicator showing current step and remaining steps
- User can navigate back and forth between steps
- Data is preserved when navigating between steps
- Step validation: cannot proceed to next step without completing required fields
- Final step: review and submit for verification
- After submission: confirmation screen with "pending verification" status
- Responsive: desktop, tablet, mobile
- Bilingual FR/AR/EN with RTL support
- WCAG AA accessible
