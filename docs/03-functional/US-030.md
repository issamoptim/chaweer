## Description

As a lawyer, I want to upload my professional documents (bar card, government ID, certifications) so that the admin can verify my credentials.

## Acceptance Criteria (Draft)

- Upload required documents: bar card (image/PDF), government-issued ID (image/PDF)
- Optional: additional certifications, diplomas
- File format: JPG, PNG, PDF
- File size limit: defined per file (e.g., 5MB max)
- Preview of uploaded documents before submission
- Ability to replace/remove a document before submission
- Documents stored securely and linked to verification request
- Upload progress indicator
- Error handling for failed uploads (retry option)
