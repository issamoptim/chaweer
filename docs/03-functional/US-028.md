## Description

As a lawyer, I want to register on Chaweer through a dedicated professional path so that I can create my professional account and start the onboarding process.

## Acceptance Criteria (Draft)

- Professional registration accessible via "Espace avocat" entry (footer link on consumer auth screens)
- Registration uses same auth methods as consumer (Google OAuth dominant + email/password)
- Account type set to "professional" (avocat) upon registration
- After registration, user is redirected to the onboarding wizard (US-033)
- Existing consumer accounts cannot be converted to professional accounts (separate registration)
- Bilingual FR/AR/EN support with RTL
