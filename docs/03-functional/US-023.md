## Description

As a user, I want to manage my account security so that I can change my password and delete my account if needed.

## Acceptance Criteria

- Three cards: Account (read-only), Password, Account Deletion
- **Account card**: displays email and authentication method badge (email/password or Google)
- **Password card (email variant)**: 3 password fields (current, new, confirm) with eye toggle; validation (min 8, letter + digit); change button with loading state; success/error toasts
- **Password card (Google variant)**: informational block — "Your password is managed by Google"
- **Deletion card**: danger-styled; button opens confirmation dialog; dialog has focus trap + Escape = cancel
- All text via i18n keys (no hardcoded strings)
- WCAG AA: keyboard navigation, focus visible, aria-live for toasts, focus trap in dialog

## Out of Scope

Email change, MFA, session management, login history, devices, passkeys, OTP, auth provider change
