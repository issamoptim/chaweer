## Description

As a user, I want to edit my personal information so that I can keep my profile up to date.

## Acceptance Criteria

- User can edit: first name, last name, phone, country, city, nationality
- Email is read-only (linked to authentication) with lock icon
- Country and city are select fields (no free text); city depends on country
- Save button is disabled until a modification is detected
- Explicit save only — no auto-save
- Unsaved changes guard: confirmation modal before leaving
- Success: confirmation screen with green checkmark → return to profile
- Error: toast + form preserved + field-level errors + focus on first invalid field
- Responsive: desktop 2 columns, mobile 1 column with full-width buttons
