## Description

As a user, I want to view my profile information and preferences so that I can review my personal details and manage my account settings.

## Acceptance Criteria

- User can view personal information (name, email, phone, country, city, nationality) in read-only mode
- User can view and change interface language (FR / العربية / English)
- User can toggle email and push notifications (SMS is "Bientôt disponible" + disabled)
- User can initiate account deletion from the danger zone
- All fields display "Non renseigné" when empty (never blank lines)
- Loading state shows skeletons; error state shows calm retry card
