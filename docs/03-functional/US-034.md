## Description

As an admin, I want to review submitted lawyer credentials and documents so that I can approve or reject their application to join the Chaweer platform.

## Acceptance Criteria (Draft)

- Admin sees a list of pending verification requests
- Each request shows: lawyer name, bar card number, bar association, uploaded documents
- Admin can view/download uploaded documents
- Admin can approve: lawyer status → "verified", profile becomes public in directory
- Admin can reject: must provide a rejection reason; lawyer status → "rejected"; lawyer can resubmit
- Admin can request additional information: lawyer status → "info requested"; lawyer receives notification
- Verification status history tracked (submitted, approved, rejected, info requested)
- All admin actions logged (audit trail)
- Related to US-082 (Professional Verification Management — the admin-side tool)
