## Description

As a user, I want to explicitly log out of my account so that I can end my session on a shared device.

## Rationale for Deferral

Basic session expiry (JWT token expiration) works. The current session is persistent by default (no "stay logged in" checkbox). An explicit logout flow with token invalidation is important for shared devices but not blocking launch. Deferred to Post-MVP Sprint 09.

## Acceptance Criteria (Draft)

- Logout option accessible from user menu / profile
- Clicking logout invalidates JWT and refresh token
- User redirected to landing authentication screen
- Session fully terminated (no lingering access)
