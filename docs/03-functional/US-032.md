## Description

As a lawyer, I want to set up my initial professional profile (photo, bio, consultation types, pricing) so that I have a minimum viable profile ready for the public directory.

## Acceptance Criteria (Draft)

- **Photo**: upload professional headshot (JPG/PNG, size limit)
- **Bio**: short professional biography (text area, character limit)
- **Consultation types**: select at least one (video, audio, chat) — video is primary for MVP
- **Pricing**: set base consultation fee per type and duration
- **Office location**: city/region (for directory location filter)
- **Contact preferences**: how clients can reach them (via platform only)
- Profile is saved as "draft" until verification is approved (US-034)
- After approval, profile becomes public
