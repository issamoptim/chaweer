# Documentation Métier — Chaweer

> **Statut :** En construction  
> **Dernière mise à jour :** 2026-07-19

Ce dossier est la **source de vérité unique** pour le domaine métier de Chaweer : glossaire, acteurs, processus, règles de gestion, modèle métier, thématiques transverses (réputation, paiements, litiges, etc.) et référentiels par profession.

Il contient uniquement de la documentation métier — **aucun détail d'implémentation technique**, aucune conception de base de données, aucune spécification d'API, aucune spécification UI/UX.

Les EPICs et User Stories référencent cette documentation au lieu de dupliquer les concepts métier.
