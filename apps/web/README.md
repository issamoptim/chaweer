# Chaweer Web

Application Web de la plateforme Chaweer.

## Stack

- React 19
- Vite
- TypeScript
- Tailwind CSS v4
- React Router v7
- TanStack Query
- React Hook Form
- Zod
- shadcn/ui
- Lucide React

## Prérequis

- Node.js 22 LTS
- npm

## Installation

```bash
npm install
```

## Développement

```bash
npm run dev
```

## Build

```bash
npm run build
```

## Tests

```bash
npm test
```

## Lint

```bash
npm run lint
```

## Variables d'environnement

Copier `.env.example` en `.env` et configurer les variables.

```bash
cp .env.example .env
```
